# 02. Задание 1 — Исправленная версия ТЗ

## Раздел ТЗ: Функционал корзины

### 1. Добавление товара

1. Пользователь может добавить товар в корзину, если товар доступен к продаже.
2. При добавлении товара система проверяет:
   - соответствие лимитам корзины, описанным в разделе 4;
   - доступный остаток товара.
3. Если товар уже есть в корзине, система увеличивает количество по существующей позиции, если итоговое количество не нарушает лимиты корзины.

### 2. Изменение количества

4. Пользователь может изменить количество товара в корзине в диапазоне от 0 до 10 единиц.
5. Если новое количество больше 0, но нарушает лимиты корзины или превышает доступный остаток, изменение не применяется, пользователь получает понятное сообщение об ошибке.

### 3. Удаление товара

6. Удаление товара из корзины выполняется при уменьшении количества позиции до 0.
7. После удаления позиция исчезает из корзины, общая сумма пересчитывается.

### 4. Лимиты корзины

8. В корзине может быть не более 5 различных товаров.
9. Суммарное количество всех единиц товаров в корзине не может превышать 20.
10. Для одной позиции нельзя установить количество больше 10.

### 5. Отображение состава корзины

11. На странице корзины отображаются:
   - наименование товара,
   - количество,
   - цена за единицу,
   - стоимость позиции (цена × количество),
   - общая сумма корзины.
12. Цены отображаются в рублях без копеек. Если расчётная цена содержит копейки, система округляет значение до целых рублей по математическому правилу округления.

### 6. Поведение цены

13. В корзине отображается актуальная цена товара из каталога.
14. Если цена изменилась после добавления товара, при следующем обновлении корзины показывается новая цена.
15. При изменении цены пользователь получает уведомление в интерфейсе: «Цена товара изменилась».

### 7. Отображение рекламы

16. В корзине может отображаться рекламный блок рекомендованных товаров.
17. Временные интервалы показа рекламы по будням: 08:00–11:00 и 18:00–21:00 по локальному времени пользователя.
18. Если рекламных материалов нет или время показа не подходит, блок рекламы не отображается.

### 8. Сообщения об ошибках

19. Для каждого типа ограничения используется отдельное сообщение:
   - «Нельзя добавить больше 10 единиц одного товара»;
   - «В корзине может быть не более 5 разных товаров»;
   - «Суммарно в корзине может быть не более 20 товаров»;
   - «Недостаточно товара на складе».
20. Общие технические ошибки отображаются нейтральным сообщением: «Не удалось обновить корзину. Попробуйте позже».
